# 文化 · 成语主题分类规范（Theme Taxonomy）

本文件与 `data/culture/idioms/idioms-index.json` 中的 `theme` 数组、以及扩展候选 `idioms-expansion-candidates-050.json` 中使用的主题标签**对齐**建议，用于筛选、检索与分批次制作详情。

## 1. 建议主题（可多选）

一条成语的 `theme` 为**字符串数组**，一般 **2–4 个**标签；过多不利于导航与打标一致性。

| 主题 | 说明与典型用法 |
| --- | --- |
| **寓言故事** | 出自寓言、神话化叙述、有教训色彩（如：狐假虎威、刻舟求剑类扩展）。 |
| **学习成长** | 与学习态度、方法、坚持相关。 |
| **做人处事** | 待人接物、合作、礼义、人际边界。 |
| **智慧思考** | 认识方法、全面看问题、避免片面。 |
| **情绪表达** | 描述心情、心理感受（AABB/成语化）。 |
| **日常描写** | 生活场景、状态、常见行为。 |
| **自然景物** | 天气、风景、环境描写。 |
| **数量夸张** | 形容多、少、大、小等夸张（人山人海、千山万水等）。 |
| **动作行为** | 看、想、动等具体动作。 |
| **课堂常用** | 教学、师生互动中常出现。 |
| **儿童适合** | 语义偏浅、例句可偏低龄。 |
| **成人适合** | 需要一定社会/抽象理解。 |
| **韩国学生易懂** | 文化迁移成本低，可与韩语对比讲解。 |
| **视野** | 眼界、格局（与井底、坐井等呼应）。 |
| **补救** | 出错后及时改正（如亡羊补牢系）。 |
| **行动** | 实践、坚持、选择。 |
| **效率** | 时间、方法、事半功倍类。 |
| **专注** | 注意力、心无二用。 |
| **习惯** | 坚持与中断、学习节奏。 |
| **变化** | 环境变了方法要变。 |
| **逻辑** | 前后一致、自洽。 |
| **语言** | 表达、论辩。 |
| **教训** | 讽喻、应吸取的经验。 |
| **方法** | 策略、办法。 |
| **态度** | 学习或处事心态。 |
| **技能** | 练习生巧、熟练。 |

> 上表可随批次增补；`idioms-index` 中已使用的细分标签（如「寓言」「做事」等）在迁移到本表时，可**映射**为表中相近项，避免同一概念多套写法。

## 2. 打标建议

- 优先**稳定、可复用**的类别，少用过窄的一次性词。
- 与 **difficulty 1–5** 可交叉参考：难义或典故深的可适当提高 `difficulty` 或加「成人适合」。
- 韩文说明需要时，可额外标「**韩国学生易懂**」+ 一个语义类（如「智慧思考」）。

## 3. 与数据文件关系

- **index**：`theme` 为数组，与候选清单、详情后续字段可以逐步统一命名。
- **扩展候选** `idioms-expansion-candidates-050.json`：只含规划项，**不**进入侧栏，直至审核写入 `idioms-index.json` 与对应 `idioms-basic-00x.json`。
